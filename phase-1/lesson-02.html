Lists & Dicts — יסודות + שימושים

1.5 שעות
רענון מ-Lesson 1

למדנו על משתנים — name = 'Shai'. עכשיו נראה איך לאחסן הרבה ערכים יחד.

מה זה List (רשימה)?

List היא אוסף סדור של ערכים, בסוגריים מרובעים. כשיש לך הרבה לקוחות, הרבה URLs, או הרבה כל דבר — list היא הבית שלהם.

list_demo.py 
clients = ["מנגו", "סופר ביץ'", "קליניקת רימון"]

print(clients)        # ['מנגו', "סופר ביץ'", 'קליניקת רימון']
print(len(clients))   # 3

אינדקסים — Python מתחיל מ-0

לכל איבר ברשימה יש אינדקס מספרי. ב-Python, הספירה מתחילה מ-0.

indexing.py 
print(clients[0])    # מנגו — הראשון
print(clients[1])    # סופר ביץ' — השני
print(clients[2])    # קליניקת רימון — השלישי
print(clients[-1])   # קליניקת רימון — האחרון (מינוס = מהסוף)

הוספת איברים ועוד כלים

list_methods.py 
# הוספת איבר בסוף
clients.append("מסעדת בר אבא")
print(len(clients))   # 4

# Slicing — חיתוך
first_two = clients[:2]   # ["מנגו", "סופר ביץ'"]
last_two  = clients[-2:]  # ["קליניקת רימון", "מסעדת בר אבא"]

# לולאה על הרשימה
for client in clients:
    print(client)
טיפ: slicing

[:5] = חמשת הראשונים. [2:] = מהשלישי ואילך. [-3:] = שלושת האחרונים. תשתמש בזה הרבה.

מה זה Dict (מילון)?

Dict הוא אוסף של זוגות key-value, בסוגריים מסולסלים. במקום "הלקוח הראשון", אתה ניגש לפי שם: client["name"].

dict_demo.py 
client = {
    "name": "מנגו",
    "traffic": 45000,
    "conversion_rate": 2.3,
    "industry": "fashion"
}

# גישה לפי key
print(client["name"])           # מנגו
print(client["traffic"])        # 45000

.get() — גישה בטוחה

אם תנסה לגשת ל-key שלא קיים עם [], תקבל שגיאה. .get() מחזיר None במקום לפצוץ.

dict_get.py 
# client["rank"] יזרוק KeyError אם rank לא קיים
# .get() בטוח יותר:
rank = client.get("rank")          # None
rank = client.get("rank", 0)       # 0 — ערך ברירת מחדל

print(f"דירוג: {rank}")            # דירוג: 0

מתי זה באמת שימושי? דמיין שאתה מקבל נתונים מ-API חיצוני, וחלק מהלקוחות לא מילאו את כל השדות:

דוגמה — נתונים חלקיים 
client = {"name": "מנגו", "traffic": 45000}   # אין conversion_rate!

# גישה ישירה — תקרוס:
# print(client["conversion_rate"])   # KeyError: 'conversion_rate'

# .get() עם ערך ברירת מחדל — לא קורס:
cvr = client.get("conversion_rate", 0.0)
print(f"CVR של {client['name']}: {cvr}%")   # CVR של מנגו: 0.0%
כלל אצבע

כשאתה לא בטוח שהשדה קיים — תמיד .get().

השוואה: List לעומת Dict

מבנה סוגריים גישה לפי מתי משתמשים
list [...] אינדקס מספרי רשימת ערכים סדורים
dict {...} key בשם אובייקט עם שדות בשם

iteration על dict — איך עוברים על כל הזוגות

לפעמים אתה רוצה לעבור על כל הזוגות key-value ב-dict. יש 3 דרכים:

דוגמה 
client = {"name": "מנגו", "traffic": 45000, "industry": "fashion"}

# 1. רק keys
for key in client.keys():
    print(key)          # name, traffic, industry

# 2. רק values
for value in client.values():
    print(value)        # מנגו, 45000, fashion

# 3. שניהם יחד — הכי שימושי
for key, value in client.items():
    print(f"{key}: {value}")
    # name: מנגו
    # traffic: 45000
    # industry: fashion
הנפוץ ביותר

client.items() — נותן זוגות (key, value) שאפשר לפרק ישירות לשני משתנים.

Functions — def, params, return

1 שעה

למה צריך פונקציות?

עיקרון DRY: Don't Repeat Yourself. במקום לכתוב את אותו קוד 10 פעמים — כותבים פעם אחת בפונקציה, ומשתמשים בה 10 פעמים.

כבר השתמשת בפונקציות

print(), input(), len(), sorted() — כולן פונקציות מובנות של Python. עכשיו תכתוב פונקציה משלך.

syntax בסיסי

functions.py 
def שם_הפונקציה(פרמטר1, פרמטר2):
    # קוד כאן
    return תוצאה

חוקים חשובים:

  • def מתחיל את הגדרת הפונקציה
  • אחרי שם הפונקציה — סוגריים עם הפרמטרים (או ריקים)
  • הקוד בתוך הפונקציה מוזז 4 spaces פנימה (indentation)
  • return מחזיר ערך — הפונקציה עוצרת שם

return לעומת print — ההבדל הקריטי

זו אחת השאלות הראשונות שמבלבלות כל מתחיל. ההבדל פשוט אבל חשוב:

  • print — מציג ערך על המסך. הוא לא מחזיר כלום שאפשר להשתמש בו אחר כך.
  • return — מחזיר ערך מהפונקציה. אפשר לשמור אותו במשתנה ולהשתמש בו.

דוגמה להמחשה:

דוגמה 
def add_with_print(a, b):
    print(a + b)        # מציג על המסך — לא מחזיר כלום

def add_with_return(a, b):
    return a + b        # מחזיר את הערך

# שימוש:
add_with_print(2, 3)    # מדפיס: 5
result = add_with_print(2, 3)
print(result)           # מדפיס: None — כי אין return!

result = add_with_return(2, 3)
print(result)           # מדפיס: 5 — return החזיר את הערך
print(result * 2)       # מדפיס: 10 — אפשר להשתמש בערך הלאה
כלל אצבע

פונקציות צריכות return. print הוא רק ל-debug או להדפסה למשתמש.

דוגמה: חישוב ציון SEO

calculate.py 
def calculate_score(traffic, cvr):
    """מחשב ציון לקוח לפי תנועה ורייט המרה."""
    return traffic * cvr

# קריאה לפונקציה
score1 = calculate_score(45000, 2.3)
score2 = calculate_score(67000, 2.9)

print(score1)   # 103500.0
print(score2)   # 194300.0

פונקציה עם dict

print_client.py 
def print_client(client):
    """מדפיס שורת סיכום על לקוח."""
    name = client["name"]
    traffic = client["traffic"]
    cvr = client.get("conversion_rate", 0)
    print(f"{name}: {traffic:,} ביקורים | המרה: {cvr}%")

client = {"name": "מנגו", "traffic": 45000, "conversion_rate": 2.3}
print_client(client)
# מנגו: 45,000 ביקורים | המרה: 2.3%
f-string ו-:, בפורמט

{traffic:,} מוסיף פסיקים כמפרידי אלפים: 45000 הופך ל-45,000. f"..." מאפשר להכניס ערכים למחרוזת עם סוגריים מסולסלים.

JSON — מה זה, ואיך קוראים מקובץ

1 שעה

מה זה JSON?

JSON (JavaScript Object Notation) הוא פורמט טקסט לחילופי מידע. כל ה-APIs בעולם כמעט מדברים JSON. הוא נראה כמו dict ב-Python — אבל הוא טקסט רגיל, ולכן עובד בכל שפת תכנות.

clients.json 
[
  {"name": "מנגו", "traffic": 45000, "conversion_rate": 2.3},
  {"name": "סופר ביץ'", "traffic": 32000, "conversion_rate": 1.8}
]

בחוץ — רשימה JSON ([...]). בפנים — אובייקטים JSON ({...}). בדיוק כמו list של dicts ב-Python.

json.load — קריאת קובץ JSON

Python כולל ספריה בשם json. json.load מקבל file object (קובץ פתוח) — לכן צריך with open(...) קודם.

read_json.py 
import json

# with open מפתח את הקובץ ושומר אותו תחת השם f
with open("clients.json", "r", encoding="utf-8") as f:
    clients = json.load(f)

# clients עכשיו הוא list של dicts ב-Python
print(type(clients))    # <class 'list'>
print(clients[0])       # {'name': 'מנגו', 'traffic': 45000, ...}
print(len(clients))     # 8
encoding="utf-8"

חובה כשיש עברית בקובץ. בלי זה Python עלול לא לקרוא נכון את האותיות.

json.dumps — כתיבת Python → JSON string

write_json.py 
import json

data = {"name": "Shai", "role": "Head of SEO"}

# ensure_ascii=False שומר על עברית. indent=2 מסדר בשורות
json_string = json.dumps(data, ensure_ascii=False, indent=2)
print(json_string)
פלט צפוי
{
  "name": "Shai",
  "role": "Head of SEO"
}

סיכום: load לעומת dumps

פונקציה מה היא עושה קלט פלט
json.load(f) קריאת JSON מקובץ file object dict / list Python
json.loads(s) קריאת JSON ממחרוזת string dict / list Python
json.dumps(d) כתיבת Python → string dict / list JSON string
json.dump(d, f) כתיבת Python → קובץ dict / list + file כותב לקובץ

לפני התרגיל — מה זה lambda?

בתרגיל בעוד רגע נשתמש בפונקציה sorted() עם משהו שנראה כך: key=lambda c: c["traffic"].

lambda היא דרך מקוצרת לכתוב פונקציה קטנה בשורה אחת.

הנה ההשוואה:

השוואה 
# הדרך הרגילה — פונקציה מלאה
def get_traffic(client):
    return client["traffic"]

# הדרך המקוצרת — lambda
get_traffic = lambda client: client["traffic"]

# שתי הגרסאות עושות בדיוק את אותו דבר

איך קוראים את זה?

  • lambda — מילת מפתח שמכריזה: "אני פונקציה קטנה"
  • client — שם הפרמטר (כל שם, תבחר)
  • : — מפריד בין הפרמטר לבין מה שמחזירים
  • client["traffic"] — הערך שמוחזר (אין צורך לכתוב return)

איפה משתמשים? בעיקר עם פונקציות שמקבלות פונקציה אחרת כפרמטר — כמו sorted(), map(), filter():

דוגמה עם sorted 
clients = [
    {"name": "מנגו", "traffic": 45000},
    {"name": "סופר ביץ'", "traffic": 32000},
    {"name": "ננו", "traffic": 67000}
]

# למיין לפי traffic — צריך לציין לפי איזה שדה
sorted_clients = sorted(clients, key=lambda c: c["traffic"])

# התוצאה: סופר ביץ' (32000) → מנגו (45000) → ננו (67000)

עכשיו אתה מוכן לתרגיל.

רמז 1 — איך פותחים את הקובץ?

Python דורש שתפתח קובץ לקריאה לפני שאתה יכול להעביר אותו ל-json.load(). השתמש ב-with open(...) as f: — with מבטיח שהקובץ ייסגר אוטומטית גם אם תהיה שגיאה.

רמז 2 — איך ממיינים רשימה לפי שדה ב-dict?

הפונקציה sorted() מקבלת פרמטר key שאומר לה לפי מה למיין. עבור רשימה של dicts, נשתמש ב-lambda (שכרגע למדנו) כדי לבחור איזה שדה לבדוק. כדי למיין מהגבוה לנמוך — תוסיף reverse=True.

רמז 3 — איך לוקחים רק את 5 הראשונים?

רשימות תומכות ב-slicing. [start:end] מחזיר חתיכה. כדי לקבל את 5 הראשונים בלבד — [:5].

פתרון מלא (רק אחרי שניסית!)
top_clients.py 
import json

# קריאת הקובץ
with open("clients.json", "r", encoding="utf-8") as f:
    clients = json.load(f)

# מיון לפי תנועה — מהגבוה לנמוך — ולקחת 5 ראשונים
top_5 = sorted(clients, key=lambda c: c["traffic"], reverse=True)[:5]

# הדפסת התוצאות
for client in top_5:
    print(f"{client['name']}: {client['traffic']:,} ביקורים")
