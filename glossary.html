בית Phase 1 מילון

מילון מושגים

המילון מתעדכן עם כל מושג חדש שאתה פוגש. עברית כשאפשר, אנגלית כשמושג הוא מילה אנגלית מקצועית.

Python (פייתון)

שפת תכנות פופולרית, פשוטה לקריאה. ה-DSL הלא-רשמי של AI ו-data. אנחנו לומדים אותה ברמת קריאה ועריכה, לא ברמת מפתח.

n8n

פלטפורמה ויזואלית לבניית workflows (זרימות אוטומציה). חופשי בקוד פתוח. מתחבר לכל API בעולם בגרירה ושחרור.

API (Application Programming Interface)

"דלת" שתוכנה אחת פותחת בשבילך כדי שתוכנה אחרת תוכל לבקש ממנה דברים. למשל ה-API של Claude מאפשר ל-Python שלך לבקש סיכומים.

JSON (JavaScript Object Notation)

פורמט טקסט מובנה לחילופי מידע בין מערכות. כל ה-APIs כמעט מדברים JSON. נראה כמו מילון ב-Python.

Git

מערכת ניהול גרסאות. שומרת היסטוריה של כל שינוי שעשית בקוד, ומאפשרת לסנכרן בין מחשבים דרך GitHub.

GitHub

שרת בענן שמארח repositories (תיקיות עם היסטוריית גיט). חינמי לפרויקטים פרטיים. גם נותן GitHub Pages — אירוח חינמי לאתרי HTML סטטיים.
