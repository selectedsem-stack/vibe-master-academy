מילון מושגים

כל המושגים שתפגוש בקורס, מסודרים לפי קטגוריה. עברית כשאפשר, אנגלית כשמושג הוא מילה מקצועית. חפש או סנן לפי קטגוריה.

🔍 46 מושגים
שפת תכנות

Python פייתון

שפת תכנות פופולרית, פשוטה לקריאה. ה-DSL הלא-רשמי של AI ו-data. אנחנו לומדים אותה ברמת קריאה ועריכה, לא ברמת מפתח.

שפת תכנות

Script סקריפט

קובץ קוד שמכיל הוראות. קובץ Python מסתיים ב-.py. מריצים עם python3 filename.py.

מושג בסיסי

משתנה Variable

"תיבה" שמכילה ערך. דוגמה: name = "Shai" יוצר משתנה בשם name שמכיל את הטקסט Shai.

מושג בסיסי

טיפוס Type

הסוג של ערך. str לטקסט, int למספר שלם, float למספר עם נקודה, bool ל-True/False.

טיפוס

str String

טיפוס של מחרוזת — טקסט. מוקף בגרשיים: "Shai" או 'Shai'.

טיפוס

int Integer

טיפוס של מספר שלם, ללא נקודה עשרונית. דוגמה: 35.

טיפוס

float

טיפוס של מספר עם נקודה עשרונית. דוגמה: 3.14.

טיפוס

bool Boolean

טיפוס בוליאני — True או False בלבד. למצבי כן/לא.

מושג בסיסי

פונקציה Function

בלוק קוד שניתן לקרוא לו שוב ושוב. print ו-input הן פונקציות מובנות של Python.

פונקציה מובנית

print

פונקציה מובנית של Python שמדפיסה ערך למסך. דוגמה: print("Hello").

פונקציה מובנית

input

פונקציה שמחכה שהמשתמש יקליד משהו, ומחזירה את הטקסט שהוקלד. דוגמה: name = input().

מושג בסיסי

הערה Comment

טקסט בקוד ש-Python מתעלם ממנו. מתחיל ב-#. בשבילך, לזכור מה הקוד עושה.

כלי מערכת

Homebrew brew

מנהל החבילות של macOS. מתקין כלי פיתוח עם פקודה אחת בטרמינל: brew install python. הסטנדרט במאק.

כלי מערכת

Terminal

אפליקציית שורת פקודה במאק. שם מקלידים פקודות ישירות למחשב. במאק: Cmd+Space → "Terminal".

כלי פיתוח

VS Code Visual Studio Code

עורך קוד חינמי של מיקרוסופט. הסטנדרט בתעשייה. תומך ב-Python דרך extension של Microsoft.

כלי פיתוח

Extension תוסף

תוסף שמוסיף יכולות לעורך הקוד. למשל extension של Python מוסיף ניתוח קוד, הרצה, debug, ו-IntelliSense.

מושג בסיסי

API Application Programming Interface

"דלת" שתוכנה אחת פותחת בשבילך כדי שתוכנה אחרת תוכל לבקש ממנה דברים. למשל ה-API של Claude מאפשר ל-Python שלך לבקש סיכומים.

פורמט נתונים

JSON JavaScript Object Notation

פורמט טקסט מובנה לחילופי מידע בין מערכות. כל ה-APIs כמעט מדברים JSON. נראה כמו מילון ב-Python.

Git

Git

מערכת ניהול גרסאות. שומרת היסטוריה של כל שינוי שעשית בקוד, ומאפשרת לסנכרן בין מחשבים דרך GitHub.

Git

GitHub

שרת ענן שמארח repositories (תיקיות עם היסטוריית גיט). חינמי לפרויקטים פרטיים. גם נותן GitHub Pages — אירוח חינמי לאתרי HTML סטטיים.

Git

Commit

פעולה ב-Git שמתעדת שמירה של שינויים עם תיאור. כמו save עם רישום היסטוריה.

Git

Push

פעולה ב-Git ששולחת commits מקומיים ל-GitHub כדי שיהיו נגישים ממחשבים אחרים.

אוטומציה

n8n

פלטפורמה ויזואלית לבניית workflows. מתחבר לכל API בעולם בגרירה ושחרור. ה-engine של פאזה 2 בקורס.

מבנה נתונים

List רשימה

אוסף סדור של ערכים, בסוגריים מרובעים. דוגמה: clients = ["מנגו", "סופר ביץ'"]. ניגשים לפי אינדקס: clients[0].

מבנה נתונים

Dict מילון

אוסף של זוגות key-value, בסוגריים מסולסלים. דוגמה: client = {"name": "מנגו", "traffic": 45000}. ניגשים לפי key: client["name"].

מושג בסיסי

Loop לולאה

חזרה על קוד מספר פעמים. ב-Python: for item in list: רץ פעם אחת על כל איבר ברשימה.

מושג בסיסי

Index אינדקס

המספר הסידורי של איבר ברשימה. Python מתחיל מ-0: list[0] הוא הראשון, list[1] השני.

מושג בסיסי

Key מפתח

השם של ערך בdict. בדוגמה {"name": "מנגו"} — name הוא הkey, "מנגו" הוא הvalue.

תקשורת רשת

HTTP HyperText Transfer Protocol

השפה בה דפדפנים ושרתים מדברים. כל קריאה ל-API היא בעצם HTTP request — בקשה ותגובה.

תקשורת רשת

Request בקשה

הצד שאתה שולח כשמבקש משהו מ-API. כולל method (GET/POST), URL, headers, ולפעמים body.

תקשורת רשת

Response תגובה

מה ש-API מחזיר אחרי בקשה. כולל status code (200/404/500), headers, ו-body (בדרך כלל JSON).

תקשורת רשת

Status Code קוד מצב

מספר תלת-ספרתי שמתאר מה קרה בקריאה. 200 = הצלחה, 404 = לא נמצא, 500 = שגיאת שרת.

תקשורת רשת

Header

מידע נוסף שנשלח עם בקשה או תגובה. למשל Authorization נושא את מפתח ה-API, Content-Type מסביר את הפורמט.

מושג בסיסי

Exception חריגה

שגיאה בזמן ריצה. למשל קריאת קובץ שלא קיים. ב-Python מטפלים בה עם try/except.

תחביר

try / except

בלוק קוד ב-Python שתופס שגיאות. הקוד ב-try רץ; אם יש שגיאה — except מטפל בה במקום לקרוס.

כלי

pip

מנהל החבילות של Python. pip install requests מתקין ספרייה ל-Python שלך.

מושג בסיסי

Library ספרייה

אוסף קוד מוכן שכתבו אחרים. במקום לכתוב כל דבר מאפס, מייבאים ספרייה ומשתמשים בפונקציות שלה.

Git

Branch ענף

גרסה מקבילה של הקוד. main הוא הענף הראשי. עובדים על ענף נפרד ואז ממזגים.

Git

Pull Request PR

בקשה למזג שינויים מענף לתוך main. ב-GitHub זה גם המקום ל-code review לפני המיזוג.

אוטומציה

Workflow

זרימת עבודה אוטומטית ב-n8n. סדרה של nodes (צעדים) המחוברים בחיצים. רץ כשטריגר מופעל.

אוטומציה

Node

צעד יחיד ב-workflow של n8n. כל node עושה דבר אחד: שולח HTTP request, מפלטר נתונים, שולח מייל, וכו'.

אוטומציה

Trigger טריגר

ה-node הראשון ב-workflow. מפעיל את הזרימה — Webhook (קריאת HTTP), Cron (זמן קבוע), Manual (לחיצת כפתור).

אוטומציה

Webhook

URL ש-n8n מספק. כל קריאת HTTP אליו מפעילה את ה-workflow ומעבירה לו את הנתונים. הדרך הכי נפוצה להפעיל workflows מבחוץ.

אוטומציה

Credential פרטי גישה

פרטי התחברות מוצפנים לשירות חיצוני (API key, OAuth token, סיסמת SMTP). מאוחסנים מוצפנים ב-n8n ומוזרקים ל-nodes בשעת ריצה.

אוטומציה

Cron

תזמון לרצות אוטומטית בזמנים קבועים. דוגמה: כל ראשון בבוקר ב-7:00. בn8n זה ה-Schedule Trigger node.

אוטומציה

Zapier

מתחרה גדול של n8n. SaaS בלבד (לא self-hosted), קל יותר לnon-developers, יקר יותר, פחות גמיש. ראשון בשוק.
